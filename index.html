# COST IQ — Οδηγός για το Laptop (SQL + Backups) Κράτα το. Όταν μπεις laptop, τρέξε με τη σειρά. --- ## 1) Ανέβασμα τελευταίας έκδοσης εφαρμογής 1. Πήγαινε **github.com/Cost-IQ/Cost-IQ** → κλικ στο **index.html** → μολυβάκι ✏️ 2. **Ctrl+A → Delete** → επικόλλησε το νέο `cloud_app.html` → **Commit changes** 3. Σε ~1 λεπτό ανεβαίνει μόνο του στο **cost-iq.netlify.app** 4. Έλεγχος: πάνω αριστερά πρέπει να γράφει **v2026.07.04·16** (ή νεότερο) --- ## 2) SQL να τρέξεις στο Supabase → SQL Editor (με τη σειρά) Τρέξε ΜΟΝΟ όσα δεν έχεις ήδη τρέξει. Αν ένα tab στην app βγάζει «Λείπει ρύθμιση βάσης», τρέξε αυτό που σου δείχνει. - [ ] `invoice_photo_cleanup.sql` — **αυτόματο σβήσιμο φωτο >12 μηνών** (κρατάει όλα τα δεδομένα) Δοκιμή τώρα (προαιρετικά): `select purge_old_invoice_photos(12);` --- ## 3) BACKUPS (αντίγραφα ασφαλείας) — πώς να είσαι κατοχυρωμένος ### Α. Αυτόματα (χρειάζεται Supabase **Pro**, ~25$/μήνα) Στο Supabase Dashboard: 1. **Database → Backups** 2. Το Pro δίνει **αυτόματα ημερήσια backups** (κρατιούνται 7 ημέρες). 3. Πρόσθετο **Point-in-Time Recovery (PITR)**: επαναφορά σε **οποιαδήποτε στιγμή** (add-on ~100$/μήνα — μόνο αν θες κορυφαία ασφάλεια για πολλά μαγαζιά). ### Β. Δωρεάν — χειροκίνητο backup (μια στο τόσο) Από το laptop, με το Supabase CLI ή pg_dump: ``` # 1. Εγκατάσταση CLI (μία φορά): https://supabase.com/docs/guides/cli # 2. Backup όλης της βάσης σε αρχείο: supabase db dump -f backup_YYYYMMDD.sql --db-url "postgresql://postgres:[PASSWORD]@db.jtakdezlvurjekyzkiox.supabase.co:5432/postgres" ``` Το `[PASSWORD]` είναι ο κωδικός της βάσης (Supabase → Project Settings → Database). Κράτα το αρχείο σε ασφαλές μέρος (Drive/δίσκο). Επαναφορά: `psql ... < backup.sql`. ### Γ. Backup μόνο των φωτο (Storage) Οι φωτο δεν μπαίνουν στο db dump. Αν θες αντίγραφο: - Supabase → **Storage → invoices** → μπορείς να κατεβάσεις αρχεία, ή - με CLI/API μαζικά (πες μου να σου δώσω script). --- ## 4) Χωρητικότητα — γρήγορη υπενθύμιση - **Δεδομένα** (τιμολόγια/τιμές): ελάχιστος χώρος → κρατιούνται **χρόνια/για πάντα**. - **Φωτο**: πιάνουν χώρο → γι' αυτό το auto-cleanup 12 μηνών. - **1 GB δωρεάν Storage** ≈ 2.000–4.000 φωτο. **Pro (100 GB)** ≈ 250.000+ φωτο. - Ενεργό το auto-cleanup + η **συμπίεση φωτο** (v16) → ο χώρος μένει σταθερά χαμηλός. --- ## 5) Λίστα «όταν μεγαλώσω σε πολλά μαγαζιά» - [ ] Multi-shop: admin διαλέγει μαγαζί, προσωπικό βλέπει μόνο το δικό του (θέλει λίγο κώδικα + δημιουργία χρηστών) - [ ] Supabase **Pro** για αυτόματα backups + χώρο - [ ] (Προαιρετικά) domain **cost-iq.gr**
